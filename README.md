# 📱 Calculator App

A sleek and functional calculator application built with **Flutter**. This project demonstrates basic arithmetic operations, state management, and a clean user interface.

## ✨ Features

* **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
* **Real-time Updates:** Instant calculation results as you type.
* **Responsive Design:** Works seamlessly on Android, iOS, and Web.
* **Clean UI:** Minimalist aesthetic with a focus on usability.

## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/moha0808/calcuatorapp.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd calcuatorapp
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
